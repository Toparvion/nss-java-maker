A sample for demonstrating NSS Java Maker working with multi session SSL log.
For this sample a couple of consequent JOSM SSL sessions were captured.
Directory contains:
* josm-out.log - JVM SSL debug log captured during JOSM start and some initial part of work;
* josm.pcapng - Wireshark capture file of the same period;
* keys.nss - NSS Java Maker output file with keys for two recognized SSL sessions.
